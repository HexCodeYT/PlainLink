# PlainLink v0.1.0

PlainLink is an early macOS release for local-first copied-link cleaning.

## Highlights

- Native macOS menu bar app.
- User-level LaunchAgent clipboard watcher.
- One-shot clipboard cleaning and restore-last-original support.
- Conservative `.plainlink` rule engine that preserves unknown parameters by default.
- Rule fixtures for safer community contributions.
- External rule-source compiler for conservative ClearURLs-compatible imports.

## Distribution Notes

This release should ship only after the macOS app zip has been signed with a Developer ID Application certificate, submitted to Apple's notary service, stapled, and verified with Gatekeeper.
