# PlainLink v0.1.0

PlainLink v0.1.0 is a developer-preview macOS release for local-first copied-link cleaning.

## Highlights

- Native macOS menu bar app.
- App icon and first-run getting-started flow.
- User-level LaunchAgent clipboard watcher.
- One-shot clipboard cleaning and restore-last-original support.
- Conservative `.plainlink` rule engine that preserves unknown parameters by default.
- Rule fixtures for safer community contributions.
- External rule-source compiler for conservative ClearURLs-compatible imports.
- Import manifests and fixture verification for generated rule files.

## Distribution Notes

Current preview builds are unsigned unless produced by `scripts/release-macos-app.sh` on a machine with Developer ID credentials. If an unsigned zip is attached to this release, label it as a developer preview and expect macOS Gatekeeper warnings.

A regular-user v0.1.0 release should ship only after the macOS app zip has been signed with a Developer ID Application certificate, submitted to Apple's notary service, stapled, and verified with Gatekeeper.
