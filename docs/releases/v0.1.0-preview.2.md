# PlainLink v0.1.0 Preview 2

PlainLink v0.1.0 Preview 2 is an unsigned technical preview for macOS testers.

## Highlights

- Native macOS menu bar app.
- User-level clipboard watcher.
- One-shot clipboard cleaning and restore-last-original support.
- Conservative `.plainlink` rule engine that preserves unknown parameters by default.
- Fixture-backed rule behavior for safer community contributions.
- Tag-aware preview artifact naming.
- macOS package smoke tests before zip creation.

## Artifact

This preview build is unsigned and not notarized. macOS Gatekeeper will warn before opening the app.

Expected artifact names:

```text
PlainLink-0.1.0-preview.2-macos-arm64.zip
PlainLink-0.1.0-preview.2-macos-arm64.zip.sha256
```

## Tester Ask

Run PlainLink for seven days and report any URL it incorrectly changes or fails to clean. Please include the dirty URL, the PlainLink output, and the expected clean URL.
